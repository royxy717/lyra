# 弦歌 (Lyra)

一个优雅的音乐可视化程序，使用 React 和 Three.js 构建。

## 功能特点

- 实时音频频谱分析
- 3D 可视化效果
- 响应式设计，适配不同设备
- 沉浸式音乐体验

## 技术栈

- React 19
- Three.js / React Three Fiber
- TypeScript
- Vite

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
